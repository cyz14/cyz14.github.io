1The Rank of a Smooth Map

Analyze the local structure of a smooth map through its rank. The rank of a smooth map \(f : N \rightarrow M\) at a point \(p \in N\) is the rank of its differentail at \(p\). Two cases are of special interests: maximal rank at a point and constant rank in a neighborhood. Let \(n = \dim N\) and \(m = \dim M\). In case \(f : N \rightarrow M\) has maximal rank at \(p\), there are three not mutually exclusive possibilities:

  1. If \(n = m\), by the inverse function theorem, \(f\) is a local diffeomorphism at \(p\).

  2. If \(n \leqslant m\), then the maximal rank is \(n\) and \(f\) is an immersion at \(p\).

  3. If \(n \geqslant m\), then the maximal rank is \(m\) and \(f\) is a submersion at \(p\).

Constant rank theorem for manifolds gives a simple normal form for a smooth map having constant rank on an open set. We obtain a criterion for a level set to be a regular submanifold, which we call the constant-rank level set theorem. As we explain in subsection 2, maximal rank at a point implies constant rank in a neighborhood, so immersions and submersions are maps of constant rank. The constant rank theorem specializes to the immersion theorem and submersion theorem. The regular level set theorem is now seen to be a consequence of the submersion theorem and a special case of the constant-rank level set theorem.

1.1Constant Rank Theorem

Suppose \(f : N \rightarrow M\) is a \(C^{\infty}\) map of manifolds and we want to show that the level set \(f^{- 1} (c)\) is a manifold for some \(c\) in \(M\). The constant-rank level set theorem can be helpful.

定理 1.1. (Constant rank theorem). Let \(N\) and \(M\) be manifolds of dimensions \(n\) and \(m\). Suppose \(f : N \rightarrow M\) has constant rank \(k\) in a neighborhood of a point \(p\) in \(N\). Then there are charts \((U, \phi)\) centered at \(p\) in \(N\) and \((V, \psi)\) centered at \(f (p)\) in \(M\) such that for \((r^1, \ldots, r^n)\) in \(\phi (U)\),

\(\displaystyle (\psi \circ f \circ \phi^{- 1}) (r^1, \ldots, r^n) = (r^1, \ldots, r^k, 0, \ldots, 0) .\) (1.1)

In the constant rank theorem, it is possible that the normal form (1) for the function \(f\) has no zeros at all: if the rank \(k\) equals \(m\), then

\(\displaystyle (\psi \circ f \circ \phi^{- 1}) (r^1, \ldots, r^n) = (r^1, \ldots, r^m) .\)

定理 1.2. (Constant-rank level set theorem). Let \(f : N \rightarrow M\) be a \(C^{\infty}\) map of manifolds and \(c \in M\). If \(f\) has constant rank \(k\) in a neighborhood of the level set \(f^{- 1} (c)\) in \(N\), then \(f^{- 1} (c)\) is a regular submanifold of \(N\) of codimension \(k\).

例 1.3. (Orthogonal group). The orthogonal group \(O (n)\) is defined to be the subgroup of \(\operatorname{GL} (n, \mathbb{R})\) consisting of matrices \(A\) such that \(A^T A = I\), the \(n \times n\) identity matrix. Using the constant rank theorem, prove that \(O (n)\) is a regular manifold of \(\operatorname{GL} (n, \mathbb{R})\).

答案. Define \(f : \operatorname{GL} (n, \mathbb{R}) \rightarrow \operatorname{GL} (n, \mathbb{R})\) by \(f (A) = A^T A\). Then \(O (n)\) is the level set \(f^{- 1} (I)\). For any two matrices \(A, B \in \operatorname{GL} (n, \mathbb{R})\), there is a unique matrix \(C \in \operatorname{GL} (n, \mathbb{R})\) such that \(A B = C\). Denote by \(\ell_C\) and \(r_C : \operatorname{GL} (n, \mathbb{R}) \rightarrow \operatorname{GL} (n, \mathbb{R})\) the left and right multiplication by \(C\), respectively. Since

\(\displaystyle f (A C) = (A C)^T A C = C^T A^T A C = C^T f (A) C\)
\(\displaystyle (f \circ r_C) (A) = (\ell_{C^T} \circ r_C \circ f) (A) .\)
\(\displaystyle f \circ r_C = \ell_{C^T} \circ r_C \circ f.\)

By the chain rule,

\(\displaystyle f_{\ast, A C} \circ (r_C)_{\ast, A} = (\ell_{C^T})_{\ast, A^T A C} \circ (r_C)_{\ast, A^T A} \circ f_{\ast, A} .\) (1.2)

Left and right multiplications are diffeomorphisms, their differentials are isomorphisms. Composition with an isomorphism does not change the rank of a linear map. Hence, in (2),

\(\displaystyle \operatorname{rk}f_{\ast, A C} =\operatorname{rk}f_{\ast, A} .\)

注意. If \(f : N \rightarrow M\) is a map that with constant rank \(k\) in a neighborhood of a point \(p \in N\), its local normal form (1) relative to the charts \((U, \phi) = (U, x^1, \ldots, x^n)\) and \((V, \psi) = (V, y^1, \ldots, y^m)\) inthe constant rank theorem can be expressed in terms of the local coordinates \(x^1, \ldots, x^n\) and \(y^1, \ldots, y^m\) as follows.

\(\displaystyle \begin{array}{rl} (y^1 (f (q)), \ldots, y^m (f (q))) & = \psi (f (q)) = (\psi \circ f \circ \phi^{- 1}) (\phi (q))\\ & = (\psi \circ f \circ \phi^{- 1}) (x^1 (q), \ldots, x^n (q))\\ & = (x^1 (q), \ldots, x^k (q), 0, \ldots, 0) \end{array}\)

As functions on \(U\),

\(\displaystyle (y^1 \circ f, \ldots, y^m \circ f) = (x^1, \ldots, x^k, 0, \ldots, 0) .\) (1.3)

1.2The Immersion and Submersion Theorems

Consider a \(C^{\infty}\) map \(f : N \rightarrow M\). Let \((U, \phi) = (U, x^1, \ldots, x^n)\) be a chart about \(p\) in \(N\) and \((V, \psi) = (V, y^1, \ldots, y^m)\) a chart about \(f (p)\) in \(M\). Write \(f^i = y^i \circ f\) for the \(i\)th component of \(f\) in the cahrt \((V, y^1, \ldots, y^m)\). The linear map \(f_{\ast, p}\) is represented by the matrix \([\partial f^i / \partial x^j (p)]\). Hencce,

\(\displaystyle \begin{array}{ll} f_{\ast, p} \operatorname{is}\operatorname{injective} \Longleftrightarrow & n \leq m\operatorname{and}\operatorname{rk} [\partial f^i / \partial x^j (p)] = n,\\ f_{\ast, p} \operatorname{is}\operatorname{surjective} \Longleftrightarrow & n \geq m\operatorname{and}\operatorname{rk} [\partial f^i / \partial x^j (p)] = m. \end{array}\) (1.4)

命题 1.4. Let N and M be manifolds of dimensions n and m respectively. If a \(C^{\infty}\) map \(f : N \rightarrow M\) is an immersion at a point \(p \in N\), then it has constant rank in a neighborhood of \(p\). If a \(C^{\infty}\) map \(f : N \rightarrow M\) is a submersion at a point \(p \in N\), then it has constant rank \(m\) in a neighborhood of \(p\).

例. While maximal rank at a point implies constant rank in a neighborhood the converse is not true. \(f : \mathbb{R}^2 \rightarrow \mathbb{R}^3, f (x, y) = (x, 0, 0)\).

定理 1.5. Let \(N\) and \(M\) be manifolds of dimensions \(n\) and \(m\) respectively.

    1. (Immersion theorem) Suppose \(f : N \rightarrow M\) is an immersion at \(p \in N\). Then there are charts \((U, \phi)\) centered at \(p\) in \(N\) and \((V, \psi)\) centered at \(f (p)\) in \(M\) such that in a neighborhood of \(\phi (p)\)

      \(\displaystyle (\psi \circ f \circ \phi^{- 1}) (r^1, \ldots, r^n) = (r^1, \ldots, r^n, 0, \ldots, 0) .\)

    2. (Submersion theorem) Suppose \(f : N \rightarrow M\) is an submersion at \(p \in N\). Then there are charts \((U, \phi)\) centered at \(p\) in \(N\) and \((V, \psi)\) centered at \(f (p)\) in \(M\) such that in a neighborhood of \(\phi (p)\),

      \(\displaystyle (\psi \circ f \circ \phi^{- 1}) (r^1, \ldots, r^m, r^{m + 1}, \ldots, r^n) = (r^1, \ldots, r^m) .\)

推论 1.6. A submersion \(f : N \rightarrow M\) of manifolds is an open map.

The regular level set theorem is an easy corollary of the submersion theorem.

\(\displaystyle \psi \circ f = \pi \circ \phi = (r^1, \ldots, r^m) \circ \phi = (x^1, \ldots, x^m)\)
\(\displaystyle f^{- 1} (c) = f^{- 1} (\psi^{- 1} (\boldsymbol{0})) = (\psi \circ f)^{- 1} (\boldsymbol{0}) = Z (\psi \circ f) = Z (x^1, \ldots, x^m)\)

The following are all examples of \(C^{\infty}\) maps \(f : N \rightarrow M\), with \(N =\mathbb{R}\) and \(M =\mathbb{R}^2\).

例 1.7. \(f (t) = (t^2, t^3) .\)

例 1.8. \(f (t) = (t^2 - 1, t^3 - t)\).

例 1.9. A one-to-one immersion that is not an embedding.

例 1.10. The union of the graph of \(y = \sin (1 / x)\) and the interval \(] 0, 1 [\).

We would like conditions on map \(f\) so that its image \(f (N)\) would be a regular submanifold of \(M\).

定义 1.11. A \(C^{\infty}\) map \(f : N \rightarrow M\) is called an embedding if

  1. it is a one-to-one immersion and

  2. the image \(f (N)\) with the subspace topology is homeomorphic to \(N\) under \(f\). (The phrase "one-to-one" in this definition is redundant, since a homeomorphism is necesarily one-to-one.)

注意. There is quite a bit of terminological confusion in the literature concerning the use of the word "submanifold". Many authors give the image \(f (N)\) of a one-to-one immersion not the subspace topology, but the topology inherited from \(f\); i.e., a subset \(f (U)\) of \(f (N)\) is said to be open if and only if \(U\) is open in \(N\). The resulting space need not be a manifold at all.

For us, a submanifold without any qualifying adjective is always a regular submanifold. A regular submanifold of a manifold \(M\) is a subset \(S\) of \(M\) with the subspace topology such that every point of \(S\) has a neighborhood \(U \cap S\) defined by the vanishing of coordinate functions on \(U\), where \(U\) is a chart in \(M\).

例 1.12. (The figure eight).

\(\displaystyle f (t) = (\cos t, \sin 2 t), - \pi / 2 < t < 3 \pi / 2.\)

\(\displaystyle g (t) = (\cos t, - \sin 2 t), - \pi / 2 < t < 3 \pi / 2.\)

定理 1.13. If \(f : N \rightarrow M\) is an embedding, then its image \(f (N)\) is a regular submanifold of \(M\).

定理 1.14. If \(N\) is a regular submanifold of \(M\), then the inclusion \(i : N \rightarrow M, i (p) = p\), is an embedding.

1.4Smooth Maps into a Submanifold

定理 1.15. Suppose \(f : N \rightarrow M\) is \(C^{\infty}\) and the image of \(f\) lies in a subset of \(M\). If \(S\) is a regular submanifold of \(M\), then the induced map \(\tilde{f} : N \rightarrow S\) is \(C^{\infty}\).

例 1.16. (Multiplication map of \(\operatorname{SL} (n, \mathbb{R})\)). The multiplication map

\(\displaystyle \begin{array}{rl} \mu : \operatorname{GL} (n, \mathbb{R}) \times \operatorname{GL} (n, \mathbb{R}) & \rightarrow \operatorname{GL} (n, \mathbb{R}),\\ (A, B) & \mapsto A B, \end{array}\)

is clearly \(C^{\infty}\) because

\(\displaystyle (A B)_{i j} = \sum_{k = 1}^n a_{i k} b_{k j}\)

is a polynomial and hence a \(C^{\infty}\) function of the coordinates \(a_{i k}\) and \(b_{k j}\). However, one can not conclude in the same way that the multiplication map

\(\displaystyle \bar{\mu} : \operatorname{SL} (n, \mathbb{R}) \times \operatorname{SL} (n, \mathbb{R}) \rightarrow \operatorname{SL} (n, \mathbb{R})\)

is \(C^{\infty}\). This is because \(\{ a_{i j} \}_{1 \leqslant i, j \leqslant n}\) is not a coordinate system on \(\operatorname{SL} (n, \mathbb{R})\); there is one coordinate too many.

\(\operatorname{SL} (n, \mathbb{R}) \times \operatorname{SL} (n, \mathbb{R})\) is a regular submanifold of \(\operatorname{GL} (n, \mathbb{R}) \times \operatorname{GL} (n, \mathbb{R})\), the inclusion map is \(C^{\infty}\) by Theorem 14; therefore, the composition \(\mu \circ i\) is also \(C^{\infty}\). Because the image of \(\mu \circ i\) lies in \(\operatorname{SL} (n, \mathbb{R})\), and \(\operatorname{SL} (n, \mathbb{R})\) is a regular submanifold of \(\operatorname{GL} (n, \mathbb{R})\), by Theorem 15 the induced map

\(\displaystyle \bar{\mu} : \operatorname{SL} (n, \mathbb{R}) \times \operatorname{SL} (n, \mathbb{R}) \rightarrow \operatorname{SL} (n, \mathbb{R})\)

is \(C^{\infty}\).

1.5The Tangent Plane to a Surface in \(\mathbb{R}^3\)

Suppose \(v = \sum v^i \partial / \partial x^i |_p\) is a vector in \(T_p N\).

\(\displaystyle 0 = \frac{d}{d t} f (c (t)) = \sum_{i = 1}^3 \frac{\partial f}{\partial x^i} (c (t)) (c^i)' (t)\)

At \(t = 0\),

\(\displaystyle 0 = \sum_{i = 1}^3 \frac{\partial f}{\partial x^i} (c (0)) (c^i)' (0) = \sum_{i = 1}^3 \frac{\partial f}{\partial x^i} (p) v^i\)

The tangent plane is defined by the equation

\(\displaystyle \sum_{i = 1}^3 \frac{\partial f}{\partial x^i} (p) (x^i - p^i) = 0.\)

例 1.17. (Tangent plane to a sphere). Let \(f (x, y, z) = x^2 + y^2 + z^2 - 1.\)

2The Tangent Bundle

For us in this book the importance of the vector bundle point of view comes from its role in unifying concepts. A section of a vector bundle \(\pi : E \rightarrow M\) is a map from \(M\) to \(E\) that maps each point of \(M\) into the fibre of the bundle over the point. As we shall see, both vector fields and differential forms on a manifold are sections of vector bundles over the manifold.

2.1The Topology of the Tangent Bundle

The tangent bundle of \(M\) is the union of all the tangent spaces of \(M\):

\(\displaystyle T M = \bigcup_{p \in M} T_p M.\)

If \(\{ A_i \}_{i \in I}\) is a collection of subsets of a set \(S\), then their disjoint union is defined to be the set

\(\displaystyle \bigsqcup_{i \in I} A_i := \bigcup_{i \in I} (\{ i \} \times A_i) .\)

\(T M\) is a set, with no topology or manifold structure. We will make it into a smooth manifold and show that it is a \(C^{\infty}\) vector bundle over \(M\). The first step is to give it topology.

If \((U, \phi) = (U, x^1, \ldots, x^n)\) is a coordinate chart on \(M\), let

\(\displaystyle T U = \bigcup_{p \in U} T_p U = \bigcup_{p \in U} T_p M.\)

At a point \(p \in U\), a basis for \(T_p M\) is the set of coordinate vectors \(\partial / \partial x^1 |_p, \ldots, \partial / \partial x^n |_p\), so a tangent vector \(v \in T_p M\) is uniquely a linear combination

\(\displaystyle v = \sum_i^n c^i \frac{\partial}{\partial x^i} |_p .\)

Let \(\bar{x}^i = x^i \circ \pi\) and define the map \(\tilde{\phi} : T U \rightarrow \phi (U) \times \mathbb{R}^n\) by

\(\displaystyle v \mapsto (x^1 (p), \ldots, x^n (p), c^1 (v), \ldots, c^n (v)) = (\bar{x}^1, \ldots, \bar{x}^n, c^1, \ldots, c^n) (v) .\) (2.1)

Then \(\tilde{\phi}\) has inverse

\(\displaystyle (\phi (p), c^1, \ldots, c^n) \mapsto \sum c^i \frac{\partial}{\partial x^i} |_p\)

and is therefore a bijection. This means we can use \(\tilde{\phi}\) to transfer the topology of \(\phi (U) \times \mathbb{R}^n\) to \(T U\).

Another way to describe \(\tilde{\phi} = (\phi \circ \pi, \phi_{\ast}) .\)

Let \(\mathcal{B}\) be the collection of all open subsets of \(T (U_{\alpha})\) as \(U_{\alpha}\) runs over all coordinate open sets in \(M\):

\(\displaystyle \mathcal{B}= \bigcup_{\alpha} \{ A|A\operatorname{open}\operatorname{in}T (U_{\alpha}), U_{\alpha} a\operatorname{coordinate}\operatorname{open}\operatorname{set}\operatorname{in}M \} .\)

引理 2.1.

  1. For any manifold \(M\), the set \(T M\) is the union of all \(A \in \mathcal{B}\).

  2. Let \(U\) and \(V\) be coordinate open sets in a manifold \(M\). If \(A\) is open in \(T U\) and \(B\) is open in \(T V\), then \(A \cap B\) is open in \(T (U \cap V)\).

We give the tangent bundle \(T M\) the topology generated by the basis \(\mathcal{B}\).

引理 2.2. A manifold \(M\) has a countable basis consisting of coordinate open sets.

命题 2.3. The tangent bundle \(T M\) of a manifold \(M\) is second countable.

命题 2.4. The tangent bundle \(T M\) of a manifold \(M\) is Hausdorff.

2.2The Manifold Structure on the Tangent Bundle

We show that if \(\{ (U_{\alpha}, \phi_{\alpha}) \}\) is a \(C^{\infty}\) atlas for \(M\), then \(\{ (T U_{\alpha}, \tilde{\phi}_{\alpha}) \}\) is a \(C^{\infty}\) atlas for the tangent bundle \(T M\). It is clear that \(T M = \cup_{\alpha} T U_{\alpha}\). It remains to check that on \((T U_{\alpha}) \cap (T U_{\beta})\), \(\tilde{\phi}_{\alpha}\) and \(\tilde{\phi}_{\beta}\) are \(C^{\infty}\) compatible.

For any \(p \in U \cap V\) there are two bases for the tangent space \(T_p M\): \(\{ \partial / \partial x^j |_p \}_{j = 1}^n\) and \(\{ \partial / \partial y^i |_p \}_{i = 1}^n\).

\(\displaystyle v = \sum_j a^j \frac{\partial}{\partial x^j} |_p = \sum_i b^i \frac{\partial}{\partial y^i} |_p\) (2.2)

Apply both sides to \(x^k\),\(y^k\), we find

\(\displaystyle a^k = \sum_i b^i \frac{\partial x^k}{\partial y^i}\)
\(\displaystyle b^k = \sum_j a^j \frac{\partial y^k}{\partial x^j}\) (2.3)

Returning to atlas, we write \(U_{\alpha \beta} = U_{\alpha} \cap U_{\beta}\), \(\phi_{\alpha} = (x^1, \ldots, x^n)\), \(\phi_{\beta} = (y^1, \ldots, y^n)\). Then

\(\displaystyle \tilde{\phi}_{\beta} \circ \tilde{\phi}_{\alpha}^{- 1} : \phi_{\alpha} (U_{\alpha \beta}) \times \mathbb{R}^n \rightarrow \phi_{\beta} (U_{\alpha \beta}) \times \mathbb{R}^n\)

is given by

\(\displaystyle (\phi_{\alpha} (p), a^1, \ldots, a^n) \mapsto \left( p, \sum_j a^j \frac{\partial}{\partial x^j} |_p \right) \mapsto ((\phi_{\beta} \circ \phi_{\alpha}^{- 1}) (\phi_{\alpha} (p)), b^1, \ldots, b^n)\)

where by (2.3) and Example 24,

\(\displaystyle b^i = \sum_j a^j \frac{\partial y^i}{\partial x^j} (p) = \sum_j a^j \frac{\partial (\phi_{\beta} \circ \phi_{\alpha}^{- 1})^i}{\partial r^j} (\phi_{\alpha} (p)) .\)

By the definition of an atlas, \(\phi_{\beta} \circ \phi_{\alpha}^{- 1}\) is \(C^{\infty}\). Therefore, \(\tilde{\phi}_{\beta} \circ \tilde{\phi}_{\alpha}^{- 1}\) is \(C^{\infty}\). This completes the proof that the tangent bundle \(T M\) is a \(C^{\infty}\) manifold, with \(\{ (T U_{\alpha}, \tilde{\phi}_{\alpha}) \}\) as a \(C^{\infty}\) atlas.

2.3Vector Bundles

The natural projection map \(\pi : T M \rightarrow M, \pi (p, v) = p\) makes \(T M\) into a \(C^{\infty}\) vector bundle over \(M\), which we now define.

Given any map \(\pi : E \rightarrow M\), we call the inverse image \(\pi^{- 1} (p) := \pi^{- 1} (\{ p \})\) of a point \(p \in M\) the fiber at \(p.\) The fiber at \(p\) is often written \(E_p\). For any two maps \(\pi : E \rightarrow M\) and \(\pi' : E \rightarrow M\) with the same target space \(M\), a map \(\phi : E \rightarrow E'\) is said to be fiber-preserving if \(\phi (E_p) \subset E_p'\) for all \(p \in M\).

例 2.5. (Fiber-preserving maps). Given two maps \(\pi : E \rightarrow M\) and \(\pi' : E \rightarrow M\), check that a map \(\phi : E \rightarrow E'\) is fiber-preserving if and only if the diagram

commutes.

A surjective smooth map \(\pi : E \rightarrow M\) of manifolds is said to be locally trivial of rank \(r\) if

  1. each fiber \(\pi^{- 1} (p)\) has the structure of a vector space of dimension \(r\);

  2. for each \(p \in M\), there are an open neighborhood \(U\) of \(p\) and a fiber-preserving diffeomorphism \(\phi : \pi^{- 1} (U) \rightarrow U \times \mathbb{R}^r\) such that for every \(q \in U\) the restriction

    \(\displaystyle \phi |_{\pi^{- 1} (q)} : \pi^{- 1} (q) \rightarrow \{ q \} \times \mathbb{R}^r\)

    is a vector space isomorphism. Such an open set \(U\) is called a trivializing open set for \(E\), and \(\phi\) is called a trivializaiton of \(E\) over \(U\).

The collection \(\{ (U, \phi) \}\), with \(\{ U \}\) an open cover of \(M\), is called a local tgrivialization for \(E\), and \(\{ U \}\) is called a trivializing open cover of \(M\) for \(E\).

A \(C^{\infty}\) vector bundle of rank \(r\) is a triple \((E, M, \pi)\) consisting of manifolds \(E\) and \(M\) and a surjective smooth map \(\pi : E \rightarrow M\) that is locally trivial of rank \(r\). \(E\) is called the total space and \(M\) the base space. We say that \(E\) is a vector bundle over \(M\). For any regular submanifold \(S \subset M\), the triple \((\pi^{- 1} S, S, \pi |_{\pi^{- 1} S})\) is a \(C^{\infty}\) vector bundle over \(S\), called the restriction of \(E\) to \(S\). We will often write the restriction as \(E|_S\).

例 2.6. (Product bundle). Given a manifold \(M\), let \(\pi : M \times \mathbb{R}^r \rightarrow M\) be the projection to the first factor. Then \(M \times \mathbb{R}^r \rightarrow M\) is a vector bundle of rank \(r\), called the product bundle of rank \(r\) over \(M\). The vector space structure on the fiber \(\pi^{- 1} (p) = \{ (p, v) |v \in \mathbb{R}^r \}\) is the obvious one

\(\displaystyle (p, u) + (p, v) = (p, u + v), b \cdot (p, v) = (p, b v) \operatorname{for}b \in \mathbb{R}.\)

A local trivialization on \(M \times \mathbb{R}\) is given by the identity map \(\mathbb{1}_{M \times \mathbb{R}} : M \times \mathbb{R} \rightarrow M \times \mathbb{R}\). The infinite cylinder \(S^1 \times \mathbb{R}\) is the product bundle of rank 1 over the circle.

Let \(\pi : E \rightarrow M\) be a \(C^{\infty}\) vector bundle. Suppose \((U, \psi) = (U, x^1, \ldots, x^n)\) is a chart on \(M\) and

\(\displaystyle \phi : E |_U \xrightarrow{\sim} U \times \mathbb{R}^r, \phi (e) = (\pi (e), c^1 (e), \ldots, c^r (e)),\)

is a trivialization of \(E\) over \(U\). Then

\(\displaystyle (\psi \circ \mathbb{1}) \circ \phi = (x^1, \ldots, x^n, c^1, \ldots, c^r) : E|_U \xrightarrow{\sim} U \times \mathbb{R}^n \xrightarrow{\sim} \psi (U) \times \mathbb{R}^r \subset \mathbb{R}^n \times \mathbb{R}^r\)

is a diffeomorphism of \(E|_U\) onto its image and so is a chart on \(E\). We call \(x^1, \ldots, x^n\) the base coordinates and \(c^1, \ldots, c^r\) the fiber coordinates of the chart \((E|_U, (\psi \circ \mathbb{1}) \circ \phi)\) on \(E\).

Let \(\pi_E : E \rightarrow M\), \(\pi_F : F \rightarrow M\) be two vector bundles, possibly of different ranks. A bundle map from \(E\) to \(F\) is a pair of maps \((f, \tilde{f}), f : M \rightarrow N\) and \(\tilde{f} : E \rightarrow F\), such that

  1. the diagram

    is commutative, meaning \(\pi_F \circ \tilde{f} = f \circ \pi_E\);

  2. \(\tilde{f}\) is linear on each fiber; i.e., for each \(p \in M\), \(\tilde{f} : E_p \rightarrow F_{f (p)}\) is a linear map of vector spaces.

The collection of all vector bundles together with bundle maps between them forms a category.

例. A smooth map \(f : N \rightarrow M\) of manifolds induces a bundle map \((f, \tilde{f})\),where \(\tilde{f} : T N \rightarrow T M\) is given by

\(\displaystyle \tilde{f} (p, v) = (f (p), f_{\ast} (v)) \in \{ f (p) \} \times T_{f (p)} M \subset T M\)

for all \(v \in T_p N\). This gives rise to a covariant functor \(T\) from the category of smooth manifolds and smooth maps to the category of vector bundles and bundle maps.

Any vector bundle over \(M\) isomorphic over \(M\) to the product bundle \(M \times \mathbb{R}^r\) is called a trivial bundle.

2.4Smooth Sections

A section of a vector bundle \(\pi : E \rightarrow M\) is a map \(s : M \rightarrow E\) such that \(\pi \circ s = \mathbb{1}_M\), the identity map on \(M\). This condition means precisely that for each \(p\) in \(M\), \(s\) maps \(p\) into the fiber \(E_p\) above \(p\). We say that a section is smooth if it is smooth as a map from \(M\) to \(E\).

定义 2.7. A vector field \(X\) on a manifold is a function that assigns a tangent vector \(X_p \in T_p M\) to each point \(p \in M\). In terms of the tangent bundle, a vector field on \(M\) is simply a section of the tangent bundle \(\pi : T M \rightarrow M\) and the vector field is smooth if it is smooth as a map from \(M\) to \(T M\).

例 2.8. The formula

\(\displaystyle X (x, y) = - y \frac{\partial}{\partial x} + x \frac{\partial}{\partial y} = \left[ \begin{array}{c} - y\\ x \end{array} \right]\)

defines a smooth vector field on \(\mathbb{R}^2\).

命题 2.9. Let \(s\) and \(t\) be \(C^{\infty}\) sections of a \(C^{\infty}\) vector bundle \(\pi : E \rightarrow M\) and let \(f\) be a \(C^{\infty}\) real-valued function on \(M\). Then

  1. the sum \(s + t : M \rightarrow E\) defined by

    \(\displaystyle (s + t) (p) = s (p) + t (p) \in E_p, p \in M\)

    is a \(C^{\infty}\) section of \(E\).

  2. the product \(f s : M \rightarrow E\) defined by

    \(\displaystyle (f s) (p) = f (p) s (p) \in E_p, p \in M,\)

    is a \(C^{\infty}\) section of \(E\).

Denote the set of all \(C^{\infty}\) sections of \(E\) by \(\Gamma (E)\). The proposition shows that \(\Gamma (E)\) is not only a vector space over \(\mathbb{R}\), but also a module over the ring \(C^{\infty} (M)\) of \(C^{\infty}\) functions on \(M\).

A section over the entire manifold is called the global section.

2.5Smooth Frames

A frame for a vector bundle \(\pi : E \rightarrow M\) over an open set \(U\) is a collection of sections \(s_1, \ldots, s_r\) of \(E\) over \(U\) such that at each point \(p \in U\), the elements \(s_1 (p), \ldots, s_r (p)\) form a basis for the fiber \(E_p := \pi^{- 1} (p)\). A frame \(s_1, \ldots, s_r\) is said to be smooth or \(C^{\infty}\) if \(s^1, \ldots, s^r\) are \(C^{\infty}\) sections of \(E\) over \(U\).

例. Let \(M\) be a manifold and \(e_1, \ldots, e_r\) the standard basis for \(\mathbb{R}^n\). Define \(\bar{e}_i : M \rightarrow \mathbb{R}^r\) by \(\bar{e}_i (p) = (p, e_i)\). Then \(\bar{e}_1, \ldots, \bar{e}_r\) is a \(C^{\infty}\) frame for the product bundle \(M \times \mathbb{R}^r \rightarrow M\).

例 2.10. (The frame of a trivialization). Let \(\pi : E \rightarrow M\) be a smooth vector bundle of rank \(r\). If \(\phi : E|_U \xrightarrow{\sim} U \times \mathbb{R}^r\) is a trivialization of \(E\) over an open set \(U\), then \(\phi^{- 1}\) carries the \(C^{\infty}\) frame \(\bar{e}_1, \ldots, \bar{e}_r\) of the product bundle \(U \times \mathbb{R}^r\) to a \(C^{\infty}\) frame \(t_1, \ldots, t_r\) for \(E\) over \(U\):

\(\displaystyle t_i (p) = \phi^{- 1} (\bar{e}_i (p)) = \phi^{- 1} (p, e_i), \quad p \in U.\)

We call \(t_1, \ldots, t_r\) the \(C^{\infty}\) frame over \(U\) of the trivialization \(\phi\).

引理 2.11. Let \(\phi : E|_U \rightarrow U \times \mathbb{R}^r\) be a trivialization over an open set \(U\) of a \(C^{\infty}\) vector bundle \(E \rightarrow M\), and \(t_1, \ldots, t_r\) the \(C^{\infty}\) frame over \(U\) of the trivialization. Then a section \(s = \sum b^i t_i\) of \(E\) over \(U\) is \(C^{\infty}\) if and only if its coefficients \(b^i\) relative to the frame \(t_1, \ldots, t_r\) are \(C^{\infty}\).

命题 2.12. (Characterization of \(C^{\infty}\) sections). Let \(\pi : E \rightarrow M\) be a \(C^{\infty}\) vector bundle and \(U\) an open subset of \(M\). Suppose \(s_1, \ldots, s_r\) is a \(C^{\infty}\) frame for \(E\) over \(U\). Then a section \(s = \sum c^j s_j\) of \(E\) over \(U\) is \(C^{\infty}\) if and only if the coefficients \(c^j\) are \(C^{\infty}\) functions on \(U\).

注意 2.13. If we replaces "smooth" by "continuous" throughout, the discussion in this subsection remains valid in the continuous category.
